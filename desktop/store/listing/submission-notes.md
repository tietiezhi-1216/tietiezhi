# Microsoft Store 审核说明

## 测试说明

应用无需注册 Tietiezhi 账号即可启动和查看界面。AI 对话功能不内置第三方模型凭据；如需完整测试，请在设置中添加审核人员自有的 OpenAI API 兼容服务地址、API Key 和模型名称。

应用不会申请管理员权限，也不会提升进程权限。

## `runFullTrust` 使用理由

Tietiezhi 是基于 Tauri 2 的本地优先桌面应用，使用 `mediumIL` 运行。智能体工作流需要在用户明确选择或由应用创建的工作区内访问文件、启动本地子进程，以及连接用户配置的 MCP stdio 服务。API Key 使用操作系统安全凭据存储。涉及敏感操作的工具受应用内权限模式和用户确认控制。

## 数据与网络

应用只会将用户输入及必要上下文发送到用户自行配置的模型服务或 MCP 服务。应用不提供内置模型服务，不出售个人数据，也不包含广告或默认遥测。完整说明见隐私政策。
