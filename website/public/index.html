功能特性

OpenAI 兼容接入

兼容 /v1/chat/completions 标准接口，填好 baseURL 与 API Key 即可对接主流中转站，无需额外适配。

密钥安全存储

API Key 存入系统钥匙串（macOS Keychain / Windows 凭据管理器），不明文落盘、不回传、不在界面展示。

实时流式对话

基于 SSE 的流式响应，逐字呈现，解析在原生层完成，等待更少、体验更顺。

跨平台桌面端

同一体验覆盖 Windows 与 macOS，原生窗口、深浅色主题跟随系统。更多能力（万物互联、Agent、多模态）持续演进中。

常见问题

铁铁汁是什么？
铁铁汁是 AI 模型中转站的官方桌面客户端，帮助你便捷地接入并使用中转站提供的 AI 能力。
支持哪些操作系统？
目前支持 Windows 10 / 11 与 macOS 13.3 及以上（含 Apple 芯片与 Intel 芯片）。
如何开始使用？
下载安装后，在「接入配置」中填入中转站的 baseURL 与 API Key，测试连通后即可在聊天页使用。
我的 API Key 安全吗？
安全。API Key 仅保存在系统钥匙串中，不会明文写入磁盘，也不会回传或在界面展示。
这是开源软件吗？
不是。铁铁汁为闭源专有软件，遵循专有许可协议。
